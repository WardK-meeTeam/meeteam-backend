    ## JSON 형식 코드

### 1. PROJECT_MY_APPLY (지원자 본인에게)

```json
{
  "type": "PROJECT_MY_APPLY",
  "data": {
    "receiverId": 6,
    "projectName": "React 프로젝트",
    "localDate": "2025-10-07"
  },
  "createdAt": "2025-10-07"
}
```

### 2. PROJECT_APPLY (프로젝트 소유자에게)

```json
{
  "type": "PROJECT_APPLY",
  "data": {
    "applicationId": 123,
    "projectId": 45,
    "receiverId": 3,
    "applicantId": 6,
    "applicantName": "김철수",
    "projectName": "React 프로젝트",
    "date": "2025-10-07"
  },
  "createdAt": "2025-10-07"
}
```

### 3. PROJECT_APPROVE (지원 승인 시)

```json
{
  "type": "PROJECT_APPROVE",
  "data": {
    "receiverId": 6,
    "projectId": 45,
    "approvalResult": "APPROVED",
    "date": "2025-10-07"
  },
  "createdAt": "2025-10-07"
}
```

### 4. PROJECT_REJECT (지원 거절 시)

```json
{
  "type": "PROJECT_REJECT",
  "data": {
    "receiverId": 6,
    "projectId": 45,
    "approvalResult": "REJECTED",
    "date": "2025-10-07"
  },
  "createdAt": "2025-10-07"
}
```

### 5. PROJECT_END (프로젝트 종료 시)

```json
{
  "type": "PROJECT_END",
  "data": {
    "projectId": 45,
    "memberId": 6,
    "projectName": "React 프로젝트",
    "occurredAt": "2025-10-07"
  },
  "createdAt": "2025-10-07"
}
```

## 1. 지원 취소 API 구현 (P0) ✅

### 배경
- 지원자가 본인 지원서를 취소할 수 있어야 함
- PENDING 상태에서만 취소 가능
- ACCEPTED/REJECTED 상태는 취소 불가

### API 스펙
| 항목 | 내용 |
|------|------|
| 엔드포인트 | `DELETE /api/v1/members/me/applications/{applicationId}` |
| 권한 | 본인만 취소 가능 |
| 조건 | PENDING 상태에서만 취소 가능 |
| 응답 | 취소된 지원서 정보 |

```java
@Override
public ApplicationCancelResponse cancel(Long applicationId, Long memberId) {
    ProjectApplication application = applicationRepository.findById(applicationId)
            .orElseThrow(() -> new CustomException(ErrorCode.APPLICATION_NOT_FOUND));

    // 본인 확인
    if (!application.isApplicant(memberId)) {
        throw new CustomException(ErrorCode.APPLICATION_CANCEL_FORBIDDEN);
    }

    // PENDING 상태 확인
    if (!application.isPending()) {
        throw new CustomException(ErrorCode.APPLICATION_CANCEL_NOT_ALLOWED);
    }

    application.cancel();
    return ApplicationCancelResponse.of(...);
}
```

## 2. 회원탈퇴 후 세션 만료 처리 확인 (P0) ✅

### 현재 구현 상태
회원탈퇴 API (`DELETE /api/v1/auth/withdraw`)에서 다음 처리가 이미 구현되어 있음:

```java
@DeleteMapping("/withdraw")
public SuccessResponse<String> withdraw(...) {
    // 1. 회원 탈퇴 처리 (isDeleted = true)
    authService.withdraw(memberId);

    // 2. 토큰 블랙리스트 추가
    authService.logout(accessToken);

    // 3. 쿠키 삭제
    accessTokenCookieProvider.deleteCookie(response);
    refreshTokenCookieProvider.deleteCookie(response);

    return SuccessResponse.onSuccess("회원 탈퇴가 완료되었습니다.");
}
```

### 확인된 보안 흐름
1. ✅ Access Token → 블랙리스트 등록 (남은 유효시간 동안)
2. ✅ Access Token 쿠키 → 삭제
3. ✅ Refresh Token 쿠키 → 삭제
4. ✅ 탈퇴 회원 재로그인 → 차단 (4/26 수정)
5. ✅ 탈퇴 회원 토큰 갱신 → 차단 (4/26 수정)

## 3. OAuth2/자체로그인 제거 및 세종대 포털 전용 인증 전환 (Major Refactoring) ✅

- 기존에는 OAuth2 (Google, GitHub) 및 이메일/비밀번호 자체 로그인 지원

```
3. Member 생성 (email: {학번}@sju.ac.kr, provider: sejong)
4. JWT 발급 → 쿠키 설정
```

### 주의사항
- **JWT 인프라는 유지됨**: Access Token, Refresh Token, 쿠키 기반 인증
- **Token Rotation 유지**: Refresh Token 갱신 시 새로운 Refresh Token도 발급
- **토큰 블랙리스트 유지**: 로그아웃/탈퇴 시 Access Token 블랙리스트 등록
- **세종대 포털 SSL 호환성**: `custom.java.security` 설정 필요 (SHA1 알고리즘)
